# Contributing

Thanks for contributing to `bdtui`.

## Prerequisites

- Go toolchain (version from `go.mod`)
- `bd` CLI available in `PATH`

## Local setup

```bash
go test ./...
go build ./...
```

## Development workflow

1. Create or pick a beads issue.
2. Implement a focused change.
3. Run tests and build locally.
4. Update beads issue notes/summary.
5. Open a PR with a concise change description.

## Commit style

- Keep commits small and focused.
- Reference beads IDs when relevant (for example: `bdtui-56i.18.3`).
