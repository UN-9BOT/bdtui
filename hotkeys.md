# Dashboard Hotkeys

## Navigation

| Key               | Action                |
| ----------------- | --------------------- |
| `h` / `←`         | Prev column (wrap)    |
| `l` / `→`         | Next column (wrap)    |
| `j` / `↓`         | Next issue            |
| `k` / `↑`         | Prev issue            |
| `0`               | First issue in column |
| `G`               | Last issue in column  |
| `Enter` / `Space` | Open details          |

## Issue Management

| Key      | Action                                   |
| -------- | ---------------------------------------- |
| `n`      | Create new issue                         |
| `N`      | Create subtask (parent = current)        |
| `b`      | Create blocked issue (current = blocker) |
| `e`      | Edit issue (inline form)                 |
| `Ctrl+X` | Edit issue in `$EDITOR`                  |
| `d`      | Delete issue (preview cascade)           |
| `x`      | Close / Reopen issue (with reason)       |

## Issue Properties

| Key | Action                     |
| --- | -------------------------- |
| `p` | Cycle priority forward     |
| `P` | Cycle priority backward    |
| `s` | Cycle status forward       |
| `S` | Cycle status backward      |
| `z` | Toggle children visibility |
| `y` | Copy issue ID to clipboard |

## Search & Filter

| Key       | Action                   |
| --------- | ------------------------ |
| `/` / `f` | Focus search             |
| `c`       | Clear search and filters |

## Relations (g-prefixed)

| Key   | Action              |
| ----- | ------------------- |
| `g B` | Blocker picker      |
| `g p` | Parent picker       |
| `g u` | Jump to parent      |
| `g P` | Clear parent        |
| `g d` | Dependencies list   |
| `g D` | Toggle dim override |
| `g o` | Toggle sort mode    |

## Herdr Integration (t-prefixed)

| Key   | Action                      |
| ----- | --------------------------- |
| `t s` | Send issue to herdr         |
| `t S` | Pick herdr target, then send |
| `t a` | Attach herdr target         |
| `t d` | Detach herdr target         |

## General

| Key | Action                     |
| --- | -------------------------- |
| `r` | Refresh (reload from `bd`) |
| `?` | Help                       |
| `q` | Quit                       |
